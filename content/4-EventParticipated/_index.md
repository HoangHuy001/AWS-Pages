# Events Attended

## Event 1 – FCAJ Community Day (First Cloud Journey Community)

**Date:** May 23, 2026

**Location:** 26th Floor, Bitexco Financial Tower, 02 Hai Trieu Street, Sai Gon Ward, Ho Chi Minh City, Vietnam

**Role:** Participant

**Overview:** A knowledge-sharing event organized for the First Cloud Journey community, focusing on Artificial Intelligence and Amazon Web Services (AWS). The speakers discussed AI mindset, Prompt Engineering, AWS cost optimization, and infrastructure security using Amazon CloudFront, AWS WAF, and SYN Proxy.

**Key Activities:**

- Learned how AI can be effectively applied in real-world scenarios.
- Explored AWS infrastructure security and cost optimization strategies.
- Listened to Hackathon experiences, with an emphasis on teamwork and solving real business problems.

**Key Outcomes and Benefits:**

- Gained a better understanding that AI is a powerful tool to assist human work rather than replace it.
- Learned the importance of Context Engineering for improving AI-generated responses.
- Understood the process of building practical software solutions while considering business requirements.
- Learned best practices for controlling AWS costs and securing cloud infrastructure.
- Strengthened teamwork skills and practical problem-solving abilities.
