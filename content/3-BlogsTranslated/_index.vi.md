### [Blog 1 - Dual-token Authentication cho Nakama Game Server với Amazon Cognito trên AWS](3.1-Blog1/)
Bài viết này giải thích cách kết hợp Amazon Cognito và Nakama Game Server bằng mô hình xác thực kép (dual-token authentication). Nội dung đề cập đến vai trò của Go runtime hook, Amazon CloudFront, ALB, NLB và ECS trên AWS Fargate trong việc xây dựng trải nghiệm chơi game trực tuyến an toàn và mượt mà.

### [Blog 2 - Hạn chế truy cập AWS Management Console bằng Sign-in Resource-based Policies và RCPs](3.2-Blog2/)
Bài viết này mô tả cách doanh nghiệp có thể giới hạn quyền đăng nhập vào AWS Management Console từ các mạng đáng tin cậy bằng Sign-in Resource-based Policies và Resource Control Policies (RCPs). Nó cũng cho thấy cách AWS Organizations và CloudTrail hỗ trợ quản trị tập trung và kiểm toán hiệu quả.

### [Blog 3 - AWS GenAIIC Partner Agent Factory – Cung cấp AI Agent qua AWS Marketplace](3.3-Blog3/)
Bài viết này giới thiệu chương trình AWS GenAIIC Partner Agent Factory và giải thích cách AWS cùng các đối tác hợp tác để cung cấp các AI agent đã được xác thực qua AWS Marketplace. Nội dung đề cập đến các dịch vụ như Amazon Bedrock, Bedrock AgentCore và Strands Agents.
